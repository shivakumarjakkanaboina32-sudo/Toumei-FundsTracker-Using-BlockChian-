# Toumei — Track Every Rupee

Toumei is a blockchain-based fund tracking system that ensures transparency and accountability in donations.

## 🚀 Features
- Donor deposits funds into smart contract
- NGO creates spending requests with proof
- Donor approves requests
- Funds are released directly to vendor
- Live event tracking

## 🛠 Tech Stack
- Solidity
- Ethers.js
- MetaMask
- Ganache
- HTML / JavaScript

## 🎬 Demo Flow
1. Deposit funds
2. Create request
3. Approve request
4. Execute payment

## 🌍 Impact
Ensures every rupee is traceable and prevents misuse of funds.
